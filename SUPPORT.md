# Support

## Getting Help

- **Questions & Discussion** — [GitHub Discussions](https://github.com/Novathes/elba-mcp-server/discussions)
- **Bug Reports** — [Open an issue](https://github.com/Novathes/elba-mcp-server/issues/new?template=bug_report.yml)
- **Feature Requests** — [Open an issue](https://github.com/Novathes/elba-mcp-server/issues/new?template=feature_request.yml)
- **Security Vulnerabilities** — See [SECURITY.md](SECURITY.md) (do NOT open public issues)

## Elba Platform Support

For issues with the Elba platform itself (not this MCP server), contact:

- 📧 support@kolsetu.com
- 🌐 [Elba Dashboard](https://elba.kolsetu.com)

## Before Opening an Issue

1. Check [existing issues](https://github.com/Novathes/elba-mcp-server/issues) to avoid duplicates
2. Make sure you're on the latest version: `npx @kolsetu-opensource/elba-mcp-server@latest`
3. Verify your integration token is valid in the [Elba Dashboard](https://elba.kolsetu.com)
